How should agent output be rendered?

Today it's raw event JSON. Here are three directions for making it readable.

A

Chat Thread

Render agent output like a conversation. Thinking blocks collapsed, tool calls as summary cards, text responses as markdown bubbles. Similar to how Claude Code looks in a terminal — but styled.

B

Timeline / Activity Feed

Vertical timeline with icons per event type. Compact, scannable, good for monitoring multiple agents at once. Thinking is a dot, tool calls are icons, text is inline.

C

Hybrid: Chat + Collapsible Detail

Default view is the chat thread (A), but each event is expandable to show raw detail. Tool calls expand to show input/output. Thinking blocks expand to show full reasoning. Best of both worlds but more complex.
